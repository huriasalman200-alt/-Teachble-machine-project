# Dolphin, Orca, and Shark Image Classification using Teachable Machine

## Project Overview
This project uses Google's Teachable Machine to build an image classification model capable of distinguishing between three marine animals:
- Orca
- Dolphin
- Shark

The trained model was exported and tested successfully using Google Colab to ensure that it works correctly and can accurately classify new images.

## Project Objectives
- Build an image classification model using Teachable Machine.
- Train the model to recognize three different marine animals.
- Export the trained model.
- Verify that the model works correctly in Google Colab.
- Upload the complete project to GitHub.

## Dataset
The dataset consists of images from three classes:
- Orca
- Dolphin
- Shark

Images for each class were uploaded to Teachable Machine and used to train the model.

## Tools Used
- Google Teachable Machine
- Google Colab
- TensorFlow / Keras
- GitHub

## Project Files
- colab code.ipynb – Google Colab notebook used to test the model.
- keras_model.h5 – The trained image classification model.
- labels.txt – Labels of the three classes.
- Teachable machine.png – Screenshot of the Teachable Machine training interface.
- colab.png – Screenshot showing the model running in Google Colab.

## Project Workflow
1. Collect images of Orcas, Dolphins, and Sharks.
2. Upload the images to Google Teachable Machine.
3. Train the image classification model.
4. Export the trained model.
5. Load the model into Google Colab.
6. Run the prediction code.
7. Test the model using sample images to verify that it correctly identifies Orcas, Dolphins, and Sharks.

## Results
The model was successfully trained and tested. It correctly distinguished between Orcas, Dolphins, and Sharks when tested in Google Colab, confirming that the exported model and prediction code work properly.

## Conclusion
This project demonstrates how Google Teachable Machine can be used to create an image classification model without building a neural network from scratch. The model was successfully exported, tested in Google Colab, and uploaded to GitHub.
